⚙️ 導入手順（4ステップ）

STEP 1：標準モジュールのインポート
  1. Excelで対象ブックを開く
  2. Alt + F11 → VBEを開く
  3. メニュー「ファイル」→「ファイルのインポート」
  4. FloatingPalette.bas を選択してインポート
STEP 2：UserForm の作成
  1. VBE のメニュー「挿入」→「ユーザーフォーム」
  2. 追加されたフォームを右クリック → プロパティ
  3. Name を frmPalette に変更（重要！）
  4. フォームをダブルクリックしてコードウィンドウを開く
  5. frmPalette_Code.bas の中身をすべてコピー&ペースト
STEP 3：ショートカットキーの設定
  1. Excelに戻る（Alt + F4 で VBE を閉じる）
  2. 「開発」タブ → 「マクロ」→ ShowPalette を選択
  3. 「オプション」→ ショートカットキーを設定（例：Ctrl + Shift + P）
STEP 4：設定シートの作成（任意）
  1. Alt + F8 → CreateSettingSheet を実行
  2. 「設定」シートが自動生成される
  3. ボタンのラベル・アクション・パラメータをカスタマイズ可能

🎯 アクション一覧

アクション名 機能 パラメータ
InsertText アクティブセルにテキスト挿入 固定テキスト（省略可）

ボタンラベル アクション パラメータ（任意）

以下、4～6行目も同様
💡 ヒント： ボタン数は最大6個（2列×3行）です。設定シートを編集したら、パレットを閉じて再起動（ShowPalette）してください。

🚀 自動起動（Excelを開くと同時にパレットが出現）

ThisWorkbook モジュールに以下を追加してください：

Private Sub Workbook_Open()
    ShowPalette
End Sub

❓ よくある質問

Q: ボタンが6個以上欲しい

現在は 2×3（6個）固定です。3×3（9個）への拡張は可能ですので、ご相談ください。

Q: 色選択をもっと便利にしたい

現在は RGB 入力形式です。Excel のカラーダイアログ連携も検討できます。

Q: Excel を閉じるとパレットも消える？

はい。再度 ShowPalette マクロを実行してください。自動起動させるには STEP で紹介した Workbook_Open イベントを使いましょう。

Q: 操作方法は？

タイトルバーをドラッグ → パレットを移動
✕ ボタン → パレットを閉じる
ショートカットキー → パレットを開く（設定後）
